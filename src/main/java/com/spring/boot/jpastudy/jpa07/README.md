# 07 `@Embeddable` 다른 테이블에 매핑하기

- @SecondaryTable
  - 다른 테이블에 저장된 데이터를 @Embeddable로 매핑 가능
  - 다른 테이블에 저장된 데이터가 개념적으로 밸류(값)일 때 사용
    - 1-1 관계인 두 테이블을 매핑할 때 종종 출현
